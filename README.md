# 形而上学

阳光保险在线学习平台（xue.sinosig.com）自动化脚本合集。

## 脚本列表

### 阳光学堂自动看视频助手

自动播放课程视频、自动考试答题的 Tampermonkey 油猴脚本。

| 版本 | 说明 |
|------|------|
| 10.3.2 | 基础版：自动播放、防切屏、考试盲答 |
| 10.3.3 | 优化考试冷却逻辑，修复卡死问题 |
| 10.3.4 | 完善正式考试答题流程，增加日志面板 |

### 阳光学习平台16倍速

强制 HTML5 视频以 16 倍速播放，绕过浏览器最高倍速限制。

## 使用方式

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开对应 `.user.js` 文件，点击安装
3. 访问 `https://xue.sinosig.com/` 即可自动运行
